## Requirements


### Git or GitHub CLI

```bash
sudo apt install git
```

or

```bash
sudo apt install gh
```


### Stow

```bash
sudo apt install stow
```

## Installation


```bash
$ git clone https://github.com/RodrigoAroeira/.dotfiles.git
```
or
```bash
$ gh repo clone RodrigoAroeira/.dotfiles
```

And then
```bash
$ cd dotfiles
```

Now, inside the folder, run

```bash
$ stow .
```
